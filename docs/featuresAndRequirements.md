# Z-Academy 5.0 - Feature List

## 1. PUBLIC FEATURES (Non-logged in users)

### Must Have:

- Home page with Hero section, Featured Courses, Trending Courses, and Categories
- Course browsing with Search and Filters (Category, Price, Level, Rating)
- Course Detail Page (Title, Description, Thumbnail, Instructor info, Rating & Reviews, Price)
- Login/Signup page (Google Authentication only)
- Public Analytics / Marketing Section (Top Courses, Most Enrolled, Highest Rated)

### Should Have:

- Category-wise course listing

---

## 2. ADMIN FEATURES

### Must Have:

- Admin Dashboard
- Verify/Reject Instructor Sign ups
- Verify/Reject Courses
- Manage Categories (Create and Delete only)
- View All Users (Students + Instructors)
- Ban/Suspend Users
- Remove Courses
- Platform Overview Analytics
- Commission Management (View 5% commission per enrollment)
- Notifications Management

### Admin Analytics should include:

- Total Revenue (Platform)
- Total Commission Earned
- Total Students
- Total Instructors
- Total Courses (Pending + Live)
- User Growth Chart (Monthly)
- Most Popular Courses
- Revenue Trend Chart
- Category-wise performance

### Should Have:

- User management (Search, Filter, View details)
- Course management (All courses list)
- Payout tracking for instructors (even if dummy)

---

## 3. INSTRUCTOR FEATURES

### Must Have:

- Google Signup + Admin Approval
- Instructor Profile (Name, Email, Avatar, Bio, Education, Experience)
- Create Course (Title, Description, Thumbnail, Video ≤20MB, Price, Category, Level)
- Course submitted for Admin Review
- Instructor Dashboard
- Manage My Courses (Edit, View Status)
- View Enrolled Students per course
- Course Public Chat management
- 1-to-1 Chat with students
- Earnings Overview (Total earned, Admin commission 5%)
- Notifications (In-app + Push)

### Instructor Analytics should include:

- Total Courses
- Total Students Enrolled (across all courses)
- Total Earnings + Admin Commission breakdown
- Course-wise performance (Enrollments, Revenue, Completion Rate, Rating)
- Top performing course
- Monthly earnings chart

### Should Have:

- Limited course editing after approval

---

## 4. STUDENT FEATURES

### Must Have:

- Google Signup/Login
- Student Profile (Full Name, Email, Avatar, Bio)
- Browse & Search Courses
- Course Enrollment (Dummy Payment)
- My Learning Dashboard (Enrolled Courses)
- Course Player (Single Video) with progress tracking (auto save watch percentage)
- Course-based Public Chat (Text + Files: image, pdf, docx, voice notes - max 2MB)
- 1-to-1 Private Chat with Instructor or other students in the same course
- Rate & Review course after enrollment
- Notifications (In-app + Firebase Push)
- Student Analytics Dashboard

### Student Analytics should include:

- Total Enrolled Courses
- Overall Progress (All courses)
- Completed Courses
- Watch time statistics
- Most watched courses
- Recent activity

### Should Have:

- Continue Watching section
- Certificate on 100% completion (optional for later)
